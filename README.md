# Chronos
A simple, fast and lightweight documentation server written in Go.

## Features
- API based response
- Markdown support
- Search
- Localized
- Git based articles storage

## Configuration
Edit the `config/chronos.json` file to configure the server.

## Run (dev)
```bash
go run main.go
```

## Build
```bash
go build -o chronos main.go
```

## Run (prod)
```bash
./chronos
```
